# Takon AI Skills Repository 🛠️

ชุดเครื่องมือ AI Agent Skills สำหรับงานวิศวกรรม สถาปัตยกรรม และการจัดการโครงการก่อสร้าง พัฒนาโดย Takon

## Skills ที่มีให้ใช้งาน

1. **DXF Data Extractor** (dxf-extractor): สกัดข้อมูล Text, Blocks, และ Polylines จากไฟล์ DXF จำนวนมาก
2. **Door Schedule Processor** (door-processor): ดึงข้อมูลแบบขยายประตูและทำความสะอาดข้อมูลอัตโนมัติ
3. **Construction Doc Manager** (construction-doc): ช่วยทำรายงานประจำวันและ Dashboard ความคืบหน้าโครงการ

## การติดตั้ง

ใช้ GitHub CLI เพื่อติดตั้ง:

`bash
gh skill install takon/takon-ai-skills
`

## ข้อกำหนด
- GitHub CLI (gh) เวอร์ชัน 2.89.0 ขึ้นไป
- AI Agent ที่รองรับมาตรฐาน agentskills.io (เช่น Gemini CLI, Copilot)
