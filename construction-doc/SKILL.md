# Construction Document Manager

Skill นี้ช่วยจัดการเอกสารประจำวันของโครงการก่อสร้าง ทั้งการทำรายงาน (Daily Report) และ Dashboard ความคืบหน้า โดยใช้ AI (Typhoon) ช่วยในการสรุปเนื้อหางาน

## ความสามารถ
- **Standardized Folders**: สร้างโฟลเดอร์สำหรับรายงานประจำวันและรูปภาพอัตโนมัติ
- **Excel Automation**: แทรกข้อมูลและรูปภาพลงใน Template Excel (รายงานประจำวัน)
- **AI Integration**: เชื่อมต่อกับ Typhoon AI เพื่อสรุปงานที่สำคัญในแต่ละวัน
- **Dashboard Generation**: สร้าง Dashboard สรุปความคืบหน้าในรูปแบบ HTML

## วิธีใช้งาน
สั่งงานกับ Agent เช่น:
- "ช่วยทำ Daily Report ของวันนี้ [YYYY-MM-DD] โดยใช้รูปภาพในโฟลเดอร์ site-photos/ และสรุปงานด้วย Typhoon AI"
- "อัปเดต Dashboard ความคืบหน้าของโครงการหน่อย"
